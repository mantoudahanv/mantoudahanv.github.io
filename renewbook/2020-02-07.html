# 新增功能

# 新增签到日历

功能路径：小程序端-我的-签到页面

功能说明：新增签到日历显示，展示月签到天数，签到获得积分，在左上角进行当日签到。签到过后日历显示已签到日期。

# 优化功能

# 会员中心订货商身份等级显示优化

优化说明：修复旧版本小程序会员中心订货商不显示身份等级问题

# 开启隐藏销量后商品分组显示销量问题优化

优化说明：修复旧版本在设置中开启隐藏销量时，在商品列表中进行批量商品分组后，小程序端依旧显示该分组销量的问题

#33 营销活动-砸金蛋及筛子第二次点击无效优化

优化说明：修复小程序后台—营销活动中，疯狂砸金蛋与骰子大王游戏点击第二次无效问题

# 商品详情页显示ＳＫＵ原价的价格

优化说明： 新增商品详情页显示商品原价
