# 新增功能

# 新增内购营销功能

更新说明： 营销设置新增内购营销功能，内购营销是基于门店员工体系下的一种促销方式，通过分享获得内购券，使用户享受折扣。

设置路径： 营销—内购营销—内购管理（图1.1）

效果路径： 小程序端—首页装修链接“企业内购”（图1.2）

（图1.1）

（图1.2）

# 新增拼团营销-老带新团功能

更新说明： 拼团营销中，新增老带新团功能。

老带新团：比如某产品价格29元，拼团价24.9元，新人价19.9元，活动开始

1·新老客户单独购买，29元不变

2·老客户团购（2人团或是多人团）,24.9元

3·老客户拉新用户一起团购，新人享受19.9元的价格，拼团成功后返24.9-19.9=5元的差价到老用户的余额里。

设置路径： 营销—拼团—发起拼团—团购类型：小团—老带新团（图2.1）

效果路径： 小程序端—成功发起拼团的商品（图2.2）

               （图2.1）

（图2.2）

# 门店配送新增配送距离设置

更新说明： 新增门店配送距离设置，商户可在门店地址中增加门店配送距离（图3.1），当用户下单超出设置的距离会提示"配送地址超出当前门店配送距离"（图3.2），未超出距离则可以正常选择门店并且下单。

设置路径： 会员—门店—右侧：门店地址—门店配送距离

（图3.1）

（图3.2）

# 开发票新增填写邮箱功能

更新说明： 启小铺后台新增购物开发票时，需要填写邮箱字段控件（图4.1）；开启后，小程序端结算页面中开具发票时，可以填写邮箱（图4.2）；商品下单后，可在商品详情中查看发票信息详情。（图4.3）

设置路径： 启小铺后台—设置—系统设置—店铺设置—发票设置中

效果路径： 小程序端—确定订单页面—开具发票中

       （图4.1）

（图4.2）

                    （图4.3）

# 新增商品详情页分享功能

更新说明： 新增商品页面整合后的分享按钮（图5.1），点击分享可选择将商品分享至好物圈、分享给朋友、分享商品海报（图5.2）；“分享给朋友”、“分享商品海报”支持上下级关系绑定，分享至好物圈可进行自定义开启或关闭。

效果页面： 单品中商品名称右侧

好物圈设置路径： 设置—系统设置—店铺设置—系统设置—微信好物圈

# 分组列表内容是否可见增加配置项

更新说明： 新增商品分组”设置分组权限后，其它会员能否看到分组中商品“配置项，当配置项选择“能”时，会员可以在店铺首页和商品列表页看到分组中商品，但无法进入详情页购买。

设置路径： 启小铺后台—商品—分类管理—商品分组

# 门店库存修改增加文件导入修改

更新说明： 门店后台新增批量修改商品，可进行模板文件下载后批量导入修改库存

设置路径： 门店后台—商品—批量修改商品

# 优化功能

# 优化小程序端订单列表可以复制单号

优化说明： 长按订单号可以进行复制，优化用户体验。

效果页面：小程序端—我的—（全部订单订单列表、待发货订单列表、待收货订单列表、退换货订单列表）

# 首页弹框增加图片尺寸

优化说明： 优化小程序首页弹框，小程序端显示效果。添加260px*260px尺寸，当图片上传为不符合系统推荐尺寸，默认讲宽度调整260px，高度自适应。

设置路径： 启小铺—设置—店铺设置—是否开启自定义活动弹框

效果路径：小程序端首页

# 购物车勾选按钮优化

优化说明： 优化购物车勾选按钮较小，会误碰到商品从而跳转到商品详情，优化加大勾选按钮

# 小程序首页定位精准到区

优化说明： 优化店铺首页部分模板，定位功能从只定位到市优化成到定位至街道

# 优化sku弹层是否展示商品库存

优化说明： 优化单品中设置隐藏库存后，在选择商品sku弹层页面中依然可以看见库存问题

# 优化万能表单装修问题

优化说明： 优化万能表单装修在首页轮播组件上显示“活动不存在或已失效”问题，优化后正常打开万能表单

# 优化运费模板设置问题

优化说明： 优化运费模板设置中，只添加门店配送并且信息填写完整时，保存出现“首件件数必须大于零”的问题；优化后正常保存。

# 优化小程序端佣金显示加载不全

优化说明： 优化部分用户小程序端佣金明显加载不全的问题

# 优化地址快速识别

优化说明： 优化小程序端，添加收货地址智能识别,优化后支持更多格式识别
