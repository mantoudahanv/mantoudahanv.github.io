# 启小铺更新文档2020.03.20

# 新增功能

TIP

第101次更新

版本号：11.0.0

# 新增退款隐藏的功能

更新说明： 开启是否允许退款时

1、小程序端待发货订单列表申请售后按钮会直接隐藏

2、小程序端待收货订单申请售后时，会将“仅退款”“退货退款”进行隐藏

设置路径： 启小铺后台—订单—售后服务—退换货设置—是否允许退款

# 新增小程序自提点定位开启功能

更新说明： 开启后，下单选择自提方式，将会根据买家当前的定位以及自提地址填写的自提距离，展示可自提的自提点选项。

设置路径： 启小铺后台—设置—物流管理—提货点管理—小程序自提点开启

# 新增微信直播设置的商品增加分销功能

更新说明： 新增新用户从直播间商品链接进入时小程序时，可根据商家自定义的上级ID进行上下级关系绑定（详情操作步骤可见直播操作手册)

设置路径： 启小铺后台—商品—出售中的商品—下载直播链接

# 新增会员注册是否需要推荐人

更新说明： 新会员进入小程序时，点击立即登录页面进行授权，需有推荐人方可注册成功

设置路径： 启小铺后台—设置—系统设置—店铺设置—会员设置—会员注册是否需要推荐人

# 新增小程序端隐藏供应商商品

更新说明： 开启总部不展示供应商商品后，小程序端商品列表页面将不展示供应商上架的商品，用户只能通过扫供应商二维码进入供应商首页。

设置路径： 启小铺后台—供应商—供应商管理—总部不展示供应商商品

# 新增商品自定义导入功能

更新说明： 新增商品自定义导入功能，商家根据模板填写商品信息可以批量导入商品

设置路径： 启小铺后台—商品—商品管理—批量修改商品

# 优化功能

# 优化订单列表中未付款删除列表

优化说明： 未付款删除列表订单状态为卖家不可删，并将底部批量删除按钮去除

# 优化订货商虚拟库存扣除规则

优化说明： 订货商虚拟库存扣除后会返给订货商差价，会员下单扣除上级订货商的虚拟库存，当交易结束后，差价会在订货商佣金中显示

# 数据包导入商品优化

优化说明： 由于淘宝接口调整，数据包导入优化重新上线

# 优化新建优惠券错误

优化说明： 优化优惠券管理中新建优惠券后保存出现“参数错误”问题

# 优化订单导出问题

优化说明： 优化导出订单格式选择E店宝时，导出的订单付款时间为空，并且无法根据付款时间进行查询问题

# 优化装修组件无法跳转

优化说明： 优化首页装修中标题组件的5、6、8样式设置链接后，小程序端点击无法跳转问题

# 优化商品销售排行会员昵称为false

优化说明： 优化商品销售排行榜中，页面展示会员昵称数据为false问题

# 优化分类置顶导航链接商品重复

优化说明： 优化分类置顶导航链接商品分类后，小程序端进入分类查看会出现重复商品

# 优化后台员工设置搜索框无效

优化说明： 优化员工设置中搜索框无效

# 优化佣金详情重复展示问题

优化说明： 优化小程序端分销中心，“我的佣金”在多次切换列表情况下，会出现数据重复展示问题

# 优化商品详情sku保存问题

优化说明： 优化在商品规格设置中，设置sku的订货商阶梯价时，只保存首条数据

# 优化微信同步地址报错问题

优化说明： 优化小程序端—收货地址中进行同步微信地址时，出现部分省直辖县级行政区划city错误问题
