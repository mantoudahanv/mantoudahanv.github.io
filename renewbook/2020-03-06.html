# 启小铺更新文档2020.03.06

TIP

第100次更新

版本号：10.9.0

# 新增功能

# 商城新零售版本供应商子店铺二维码

更新说明：供应商后台新增店铺二维码

查看路径：供应商后台——店铺——店铺信息

# 展示订货商购买的虚拟库存

新增说明： 订货商购买了虚拟库存后，希望可以在订货商中心里展示出有多少库存

查看路径： 小程序端——订货商中心——订货商库存

# 种草社区

更新说明： 启小铺推出种草社区模块，商家可以通过社区来进行用户运营和内容运营，更好的提升用户购买的转化率。用户也能进行商品分享的体验，互相种草提问，调动用户积极性。

设置路径： 启小铺后台——种草生活——种草社区

展示路径： 小程序端——首页链接至种草社区

# 接入微信直播

更新说明： 启小铺新增接入微信小程序直播功能，商家可根据指引文档进行小程序直播插件的申请开通及设置。

设置路径： 启小铺后台——小程序——微信小程序——微信直播

# 分享营销

新增说明： 新增分享营销功能，设置配置项后，用户可将指定商品链接分享，可得到余额奖励

设置路径： 启小铺后台——营销——吸粉营销——分享营销

# 优化功能

# 优惠券兑换中心页面加载不全

优化说明： 优化了小程序端优惠券兑换中心，优惠券加载不全面的问题。

# 发货通知消息无通知

优化说明： 优化了订单发货后，微信没有推送发货通知消息的问题

# 短信群发发送失败

优化说明： 优化了部分用户使用短信群发功能，通过导入手机号发送失败问题

# 编辑优惠券/新增优惠券，预览金额不展示

优化说明： 优化了旧版本新增优惠券时，填写面值不会同步到预览图中
