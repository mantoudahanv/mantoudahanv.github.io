# 启小铺更新文档2020.01.03

TIP

第97次更新

# 新增功能

# 新增短信群发功能

商城后台路径：营销—群发—短信群发

更新说明：

短信群发：商家可选择平台会员及导入手机号两种方式进行短信群发

短信群发记录：可以查看历史群发短信记录

# 供应商后台新增商品码

路径：供应商后台—商品—商品码

更新说明：供应商后台增加小程序二维码获取的功能

商品码展示效果：

# 新增“营销-游戏营销”中奖奖项为赠送商品时可以设置是否需要运费

商城后台路径：营销—游戏营销—小游戏

支持的小游戏：幸运大转盘、疯狂砸金蛋、好运翻翻看、骰子大王

更新说明：新增赠送商品是否免邮选项。当按钮为是时，结算时需要支付邮费。

小程序端展示：

# 新增万能表单功能

商城后台路径：营销—吸粉营销—万能表单

更新说明：新增万能表单可用于商家在小程序内进行用户调研。默认包含字段：姓名、昵称、邮箱、微信号、手机号、QQ号。商家可自定义模块名称以及是否必填。

小程序端展示：

# 限时打折，秒杀营销活动编辑页面增加立即开启的选项

商城后台路径：营销—吸粉营销—限时秒杀/限时打折

更新说明：限时秒杀/限时打折在编辑页面都新增配置项：是否立即开启。

当勾选否时，需要在限时秒杀/限时打折活动界面手动开启。

# 门店收款码新增积分抵扣的功能

商城后台路径：会员—门店—门店收款码—编辑

更新说明：商家可自定义门店收款码积分抵现是否开启以及抵现百分比设置。

注：跟随系统设置路径为 营销—积分抵现

# 代理中心增加团队会员入口

更新说明：商城后台会员中心装修，代理中心增加团队会员入口

商城后台路径：店铺—会员中心装修—代理中心

小程序路径展示：代理商中心—团队会员

# 新增【买家留言】自定义功能设置

商城后台路径：设置-店铺设置-购物设置-买家留言

更新说明：新增买家留言是否必填的设置，买家留言提示信息自定义设置

小程序端勾选买家留言必须开启展示：

# 导出订单新增订单创建时间筛选框

商城后台路径：订单—订单管理—导出订单—订单创建时间

更新说明：商家可根据订单创建时间进行订单导出筛选

# 优化功能

# 商品选择门店自提功能操作优化

商城后台路径：商品—已经设置门店自提的商品—查看门店归属

优化说明：选择商品，点击门店自提，如果不勾选门店，则会清空当前门店归属

可直接在门店归属中删除当前所属门店

隐藏滚动条，换成一条线隔开，如果数据变多之后，滚动条才展示

# 优化限时秒杀/限时打折开启活动提醒

商城后台路径：营销—吸粉营销—限时秒杀/限时打折

优化说明：当限时秒杀/限时打折设置的活动时间结束后，点击开启活动，会提示活动过期提醒

# 优化秒杀活动购买规则

优化说明：商品详情点击选择规格，再进行购买，也可以使用秒杀价进行够买

# 分享功能优化

更新说明：

（1）营销-活动营销-游戏营销，游戏详情页面可以分享，并且可以绑定上下级

（2）营销-吸粉营销-万能表单，小程序打开万能表单详细页可以分享，并且可以绑定上下级
