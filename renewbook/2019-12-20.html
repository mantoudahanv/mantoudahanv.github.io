# 启小铺更新文档2019.12.20

TIP

第96次更新

# 新增功能

# 新增自定义活动弹框新增多种跳转模式

路径：商城小程序后台——设置——系统设置——店铺设置活动弹框跳转到

新增模式为：专题页面、分销申请、代理商申请、优惠券、拆红包链接。

内容说明：

1.  专题页面需要先到  小程序后台—店铺—自定义专题—分销专题 中设置专题才能保存。
2. 优惠券需要先到 营销—优惠营销—优惠券管理中设置后才能保存。

   ![图片](https://uploader.shimo.im/f/4TYLMM5YvJYBUvRz!thumbnail)

# 新增单品SKU重量设置

路径：小程序商城后台—商品—发布商品—库存规格—规格设置—重量

内容说明：勾选商品规格的颜色和尺码后会显示对应的商品规格，新增sku重量设置。

方便使用运费模板时按重量计价

# 店铺首页装修新增置顶自定义滑动导航组件

路径：商城小程序后台—店铺—店铺主页—编辑—分类置顶导航

# 新增订单详细页展示用户微信头像和昵称

路径：商城小程序后台—订单—所有订单详情页

# 小程序后台新增两套可编辑的行业官网模板

路径：商城小程序后台—行业官网—首页装修—可使用的模板

内容说明：点击编辑后可以自定义的使用组件装修编辑官网。如同店铺编辑

# 新增二级分类商品展示

路径：商城小程序后台——商品——分类管理——商品分类——二级分类商品展示

内容说明：新增商品分类的展示形式

# 新增首页装修链接类型：商品分类页

路径：商城小程序后台—店铺——店铺主页——编辑——组件——链接到——商品分类页

可使用的组件：图片导航，图片广告，橱窗，滑动导航

# 新增下单付款后送优惠券

路径：商城小程序后台—设置——店铺设置——购物设置——付款成功是否立即发放优惠券

内容说明:

付款成功是否立即发放优惠券开启，下单付款满足赠送规则就送优惠券；

付款成功是否立即发放优惠券不开启，下单付款不送优惠券，交易完成满足赠送规则送优惠券。

赠送规则设置一：营销——优惠营销——购物送优惠券

赠送规则设置二：商城小程序后台—商品—商品编辑—物流及其他—消费赠送优惠券

# 积分兑换商品新增每人每天只能兑换一件

路径：商城小程序后台—营销—积分营销—积分兑换商品—添加商品/编辑

# 新增门店收款码

路径：商城小程序后台—会员—门店—门店收款码

内容说明：新增门店收款码，方便门店线下收款，会员可以线下扫码支付

收款记录中可查看所有门店的订单记录

# 店铺装修单列商品新增式样三

路径：商城小程序后台—店铺主页—编辑—组件：商品

# 新增招商经理管理

路径：商城小程序后台—供应商—供应商管理—招商经理管理

# 新增订货商虚拟库存

（1）路径：商城小程序后台—会员—订货商—虚拟库存设置

内容说明：商家可以进行供应商虚拟库存设置

（2）路径：商城小程序后台—会员—订货商—订货商管理—订货商虚拟库存详情

内容说明：新增供应商虚拟库存详情，可对供应商虚拟库存进行手动管理及订货商虚拟库存记录查看。

# 订货商中心新增下级会员，团队订单

路径：小程序端—我的—订货商中心—下级会员/团队订单

内容说明： 下级会员：展示直接下级与间接下级

      团队订单：展示所有该订货商可以拿到佣金的订单

# 新增满赠营销增加允许自主选择赠品数量

内容说明：优化了赠品列表展示方式

更新前： 赠品数量分列展示

更新后：优化了赠品列表,可以自主选择赠品数量

# 新增微聊对接优惠券

内容说明：在微聊会话页面右侧的优惠券列表，显示商城小程序后台添加的优惠券，客服发送给会员，会员可领取使用

# 微软对接订单列表

内容说明：在微聊会话页面右侧的优订单列表，可直接查看订单信息。

内容说明：原版小程序模板消息升级为订阅消息

# 优化代言名片的生成清晰度

路径：小程序端—我的—代言名片

内容说明：小程序端生成代言名片后，提高小程序码展示清晰度。

# 优化商品详细页面，商品图片可以放大保存

内容说明：小程序端展示
