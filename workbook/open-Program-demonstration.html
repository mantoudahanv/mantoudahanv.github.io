# 十三.小程序端

# 13.1商品详情页

# 13.1.1购物车显示数量

操作步骤：
小程序端--选择商品，商品详情页-购物车，展示购物车中所有商品数量，如图：

# 13.1.3购物车中下单多个sku数量加起来满足起卖数即可下单

如：商品设置起卖数是10件，在购物车中结算，三个sku的数量加起来等于10即可下单，如图：
商品设置起卖数后，商品有多个sku，直接下单时一个sku要满足起卖数才能下单；先加入购物车，在购物车中下单，多个sku数量加起来满足起卖数即可下单。
加入购物车时，可以加入1件商品。

# 13.1.4商品详情展示库存

编辑商品，是否显示库存配置项，如图：

设置完成后，小程序端商品详情页会显示商品库存，如图：

# 13.1.5购物车展示商品起卖数

设置了起卖数的商品，小程序端把商品加入购物车中，购物车中对应的商品，会展示起卖数量。如图：
如果商品起卖数是5件，加入购物车时选择了4件，加入购物车后，商品的数量会自动变成起卖数的数量。

# 13.1.6 商品详情页面分享

新增商品页面整合后的分享按钮，点击分享可选择将商品分享至好物圈、分享给朋友、分享商品海报；三种分享都支持上下级关系绑定，分享至好物圈可进行自定义开启或关闭。


# 13.3商品生成海报显示对应的活动类型以及活动价

以砍价商品为例：
生成海报时，展示是什么活动商品，海报中展示的商品价格是砍价活动中设置的砍价起始价格。

备注：所有活动商品生成海报的价格，就是商品详情页展示的价格。
注：商品在活动期间，如果生成过海报之后，再修改活动中的价格，会引起生成海报价格不一致的情况，请勿随意修改活动中的价格或者折扣。
