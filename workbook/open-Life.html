# 十二.种草生活

# 12.1商城后台设置

路径：启小铺后台——视频卖货——种草社区

# 12.1.1社区设置



种草社区相关的配置项处理地方（图1）

发布文章需要审核：
开启：用户发布文章后，需要商家在启小铺后台——文章管理——待审核中进行审核；
不开启：则自动发布
评论文章需要审核：
开启：用户评论文章，需要商家在启小铺后台——文章管理——文章详情中同意；

不开启：则自动发布
评论问答需要审核：
开启后，用户评论回答，需要商家在启小铺后台——问答管理——问答详情中同意；

不开启：则自动发布
管理员昵称及头像：
修改管理员在“文章管理”、“问答管理”中发布内容后显示的头像和昵称

# 12.1.2文章管理


文章管理（图2）

发布文章：


操作步骤（图2.1）
1、点击发布文章
2、按照格式填写内容，除发布内容为必填项，其他都是选填
3、发布成功后展示在文章管理中，小程序端“种草”页面同步更新（图）
注：第2步中，选择标签需要在“文章标签”中设置标签后，再进行选择
分类展示：


展示类目（图2.2）

  • 展示中

显示正在小程序端展示的文章，商家和用户发布的同时展示

  • 总部发布

显示正在小程序端展示的文章，商家发布

  • 用户发布

显示正在小程序端展示的文章，用户发布

  • 待审核

用户提交文章发布后，商家需要审核（需要在“社区管理”中开启“发布文章需要审核”配置项）

  • 隐藏中

商家和用户所有隐藏中的文章

  • 已拒绝

在待审核文章列表拒绝过的文章

文章标签

文章标签（图3）
添加标签：

添加标签步骤（图3.1）

1、文章标签中添加新标签
2、填写标签信息
3、保存后在文章管理——发布文章中可见
4、保存后在小程序端写文章——选择标签中可见

# 12.1.3问答管理


问答管理（图4）

发布提问：

发布提问操作步骤（4.1）
1、点击发布提问
2、按照格式填写内容，除发布内容为必填项，其他都是选填
3、发布成功后展示在问答管理中，小程序端“问答”页面同步更新（图）

# 12.1.3问答分类


问答分类（图5）

1、问答分类中添加新分类
2、填写分类信息
3、保存后在问答管理——发布提问中可见
4、保存后在小程序端写问题——选择分类中可见

# 12.2小程序端

种草社区主要分为关注、种草、问答三大模块。

# 12.2.1发布

发布文章：

  • 种草和问题界面右下角都有发布按钮，点击可以选择发布内容
  • 点击“写文章“进入文章编辑页面，根据要求进行文章填写
  • 按照要求填写好文章内容后发布

发布问答：

  • 种草和问题界面右下角都有发布按钮，点击可以选择发布内容
  • 点击“提问“进入问题编辑页面，根据要求进行内容填写
  • 按照要求填写好问题后发布

# 12.2.2种草

种草页：
展示所有已发布的内容、点赞数量；
用户对已发布的内容，进行浏览、点赞、搜索、进入文章详情。


文章详情页：
展示文章图片、标题、标签、关联的商品、评论详情；
用户可进行关注、发表评论、回复评论、点赞、分享、进入关联的商品购买。


# 12.2.3问答

问答页：
展示已公开的所有问题及问题分类
用户可根据问题分类进行查看问题，并可以进入问题详情页


问题详情页：
展示问题标题、正文、图片及历史回答
用户可点击“写回答”完成互动


关注
关注页面展示关注过的用户发的动态，可进行转发、点赞、评论

个人中心
点击左上角头像可进入个人中心，可以查看赞过的文章和历史发布过的文章、问答；
并可以编辑个人资料。
