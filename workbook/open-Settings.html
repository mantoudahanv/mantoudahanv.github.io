# 9.1.2佣金设置

设置分销商推荐购买后的返佣比例、分销商的上级分销商能拿到的返佣比例、分销商上级的上级能拿到的返佣比例。佣金比例不能超过50%。如果开启了“按照利润计算佣金”，那么佣金比例则不受限制，可设置100%。开启运费参与返佣，那么运费则参与返佣。

分销内购，启用后，分销商购买商品自己可以拿一级佣金，自己的直接上级拿二级佣金；未启用时，分销商购买商品时自己是拿不到佣金的，自己的上级可以拿一级佣金。

# 9.1.3 通知消息设置

通知发货短信的意义除了在于提醒买家注意收货加快回款速度，同时还能提升店铺信誉度达到留住客户的效果，而批量发送短信则是在保证服务的情况下为商家减轻工作量，节省大量时间成本。

操作步骤：
①首先在设置→店铺设置中增加商家联系电话。

②设置→通知消息设置，以买家已付款为例：
设置消息内容，开启短信发送。

③设置→短信记录，确保当前有可用短信数量
买家手机端下单且已付款，卖家会收到买家下单短信。
后台可以查看短信发送记录。

# 9.1.4 提现设置

余额提现。开启后，会员余额可以提现。
佣金转余额。开启后，佣金可以转余额。
佣金仅限转余额。开启后，佣金只可以转到余额去消费，不能提现。
商家可设置提现方式，也可设置提现最低金额，达到该金额后才可提现，同时也可根据需求设置提现支持的账户。

后台提现设置可以设置佣金转余额不受限制，比如商家设置了佣金整额提现，开启了佣金转余额不受限制，没有达到整额的佣金也可以转到余额里面进行消费。

# 9.6 管理及权限

# 9.6.1 管理员

点击“权限角色”进行操作人员的角色划分并保存。

# 9.6.2 权限角色

权限角色划分好以后，进入到管理员，进行添加管理员的操作，并选择相应管理员的权限。

备注：微分销与小程序同一个权限角色，权限不冲突
微分销和小程序互通，微分销添加了权限角色，勾选了部分权限，进入启小铺后台，仅有该权限角色，权限为空，权限可以重新勾选，保存后，跟微分销设置的不冲突。
