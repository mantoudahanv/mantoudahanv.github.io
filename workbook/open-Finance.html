# 6.1.1提现申请管理

提现申请里，涉及资金发放，请认真核对后发放。如审核不通过，可以填写驳回的相关原因。

# 6.1.2佣金转余额审核列表

分销商可以申请把相应的佣金转到余额里面进行消费，佣金转余额审核列表记录所有分销商申请转余额的相关记录。

# 6.3 佣金记录

# 6.3.1 分销商佣金

可以看到所有分销商的一个佣金状态，根据昵称/手机号/会员ID，剩余佣金，进行查询。
可以根据昵称/手机号/会员ID，剩余佣金，进行导出。
可以根据佣金的降序和升序进行排序，可以点击详情查看当前用户的个人信息。

# 6.3.1 代理商佣金

可以看到所有代理商的一个佣金状态，根据会员ID，代理商等级，进行查询。
可以根据昵称/手机号/会员ID，剩余佣金，进行导出。
可以根据佣金的降序和升序进行排序，可以点击详情查看当前用户的个人信息。


# 6.3.1 门店佣金

可以看到所有门店的一个佣金状态，根据昵称/手机号/会员ID，剩余佣金，进行查询。
可以根据昵称/手机号/会员ID，剩余佣金，进行导出。
可以根据佣金的降序和升序进行排序，可以点击详情查看当前用户的个人信息。

# 6.2.3 账号充值记录

所有的账号充值记录，可以根据电话号码，用户账号，支付日期进行查询账号充值记录。
