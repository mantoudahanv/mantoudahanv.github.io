# 十.小程序设置

# 10.1微信小程序

# 10.1.1小程序设置

需要登录微信公众号后台注册小程序，具体操作流程参考指引



可以邀请体验者，需要后台绑定体验者的微信账号，绑定成功后，微信确定授权后扫描二维码即可进入微信小程序，如下图所示：

绑定成功后必须重新扫码才可进入商城小序程，如下图所示：

解绑体验者，输入已绑定的体验者的微信账号。

先上传代码，在提交审核，审核通过了之后才能发布小程序。

**自动上传审核：**为保证体验最新的小程序，系统将会在每次更新版本的时候，自动上传代码并提交审核，默认开启，如不需要，可关闭。如果正在审核中，此时自动上传失败。
**自动发布：**当审核状态从“审核中”改变为“审核通过”，系统会自动发布，不需要商家手动发布。
注：审核之前店铺主页、店铺设置一定要编辑，否则审核不会通过
**解除授权：**①小程序公众后台→设置→第三方服务，点击停止授权，如下图：

②回到启小铺后台→设置→小程序设置，出现“立解解除”按钮。如下图：

③点击立即解除，展示解除授权声明，只有勾选“我已知晓并同意声明才能彻底解除授权，如下图：

# 10.1.2 支付设置

微信小程序支付方式设置，开启微信付款需要到 设置—微信支付 中进行配置

# 10.1.3跳转设置

操作步骤：
开启配置项“开启小程序跳转功能”，填写要跳转的小程序号appid，最多可以添加10个小程序。如图：
（可跳转到另外的小程序）


配置完成后，在会员中心装修页面，把程序跳转“显示”勾选上

在小程序端，会员中心可以看到程序跳转入口，点击“跳转程序”--允许跳转，即可跳转到另一个小程序，如图：


注：必须是同一个公众号下，而非同个open账号下
微信官方对小程序跳转功能做了调整，需要重新调整小程序跳转的规则，请知悉，具体的调整规则请看--微信官方公告

目前启小铺跳转规则设置为只允许设置一个，后期会根据微信官方跳转个数规则进行调整，请知悉
注：此处改动之后需要重新手动上传版本，审核，审核通过发布之后才可以实现跳转设置的小程序appId

# 10.2 字节小程序

# 10.2.1 基本设置

可对字节小程序进行绑定管理，详细的字节跳动小程序上传代码,提交审核步骤可查看页面上的文档
