# 7.1.7 我要送礼

适用于包邮的商品。
拍下后直接发货到收礼人手中，收货地址是由收礼人完成的，所以在一次交易中，会有两个会员参与，如此一来可以快速的引流。送礼人（买家）可以挑选好商品并付款（此时需要再增加一个订单状态：待完善收货地址，付款后不能叫已付款或待发货，等收礼人填写完地址后才可以成为已付款或待发货状态），将该礼单分享给小伙伴（朋友圈，微信群，微信好友）由其填定收货地址并领取。
商家可以选择送礼功能是否开启，启用送礼功能后，送礼人（买家）可以挑选好商品并付款，将该礼单分享给小伙伴（朋友圈、微信群、微信好友），由其领取并填写收货信息。
    为了避免当送礼人购买了多件商品，收礼人将礼品转发给朋友，让其他人继续领的情况，商家可以设置“收礼是否需要验证码”，选择是，领取礼品的时候需要填写验证码，一个验证码对应一个商品。
送礼功能只支持包邮商品：商品编辑时，运费设置选择“包邮”即可。

点击“新建内容”按钮，添加送礼人配置，进如下图所示：

添加后保存，自动返回到我要送礼的主页面，同理可以设置收礼人配置、领取失败配置和送礼配置。

后台开启“我要送礼功能”和“收礼需要验证码”功能，商品运费设置为“包邮”：手机端买家购买我要送礼的商品，步骤如下：

# 7.4拼团活动

# 7.4.1 添加拼团活动

参与团购的商品不能享受其他优惠活动，拼团不成功，买家支付的货款会自动退回到买家余额，对微信支付的账户，拼团失败后，退还给买家会员的金额可以自动原路返回，不需要商家打款，这样能减少卖家提现的手续费（这个原路返回只针对微信支付的账户）
操作步骤：
启小铺后台--营销--拼团活动管理，发布活动

# 7.3.1.1小团

点击“添加拼团活动”进行活动规则的设置，团购类型有：小团

小团：邀请才可成为团友，邀请人为团长，满足人数才可开团。为了避免团长开团后长期不能成团，开团24小时后如果还未成团，将会自动关闭订单。小团可以设置团长是否免单，选择允许，那团长下单时支付0元就参团成功。小团也可以在活动详情页面参与别人的团团长免单的话，团长开团时只能支持购买一件哦。
开团等级：若选择了开团等级，只有达到该等级的会员才可以开团


拼团订单付款后，跳转到邀请好友参团页面

好友点击分享页面进入，点击立即参团就可以参与团购活动。



单独设置每个团购的分享标题
可自主编辑分享的拼团商品的标题，分享后，手机端也会有显示。

◆ 别人在开团
可以查看谁在开团，最多只显示两条记录（按倒计时最近的显示）

# 7.3.1.1.2老带新团

老带新团是小团中的一种，在小团类型中选择老带新团，可以进行老带新团配置。


**老带新团：**比如某产品价格29元，拼团价24.9元，新人价19.9元，活动开始
1·新老客户单独购买，29元不变
2·老客户团购（2人团或是多人团）,24.9元
3·老客户拉新用户一起团购，新人享受19.9元的价格，拼团成功后返24.9-19.9=5元的差价到老用户的余额里。

**只允许新人参团：**选择是的情况下，允许新人参团；2·新人老人都可以发起拼团，且发起拼团的价格都是新人价。

# 7.3.1.3阶梯团

# 7.11.1内购设置


**内购设置：**可以对内购营销的内购券的获得方式进行全局配置。
新客奖励：首次进入门店可获得对应门店的新客奖励
每日登录：每日进入对应的门店即可获得对应数量门店内购券
每日分享：每日分享给好友，好友点击后，好友和你均可获得对应数量的门店内购券
邀请奖励：分享的好友为首次进入该门店，即可获得该门店的邀请奖励

# 7.11.2内购管理



**添加门店:**添加门店会出现弹框

**门店名称：**下拉框选择门店；
**默认员工：**下拉框选择员工。门店员工设置可以看（4.8.1设置员工）。
内购券设置：跟随系统为内购券设置跟7.11.1内购设置中系统全局的内购券设置一致。另外也可以进行自定义设置。


添加好之后可以看到门店信息、默认员工ID、内购券设置、统计、创建时间详情。在商品首页链接后可以进入内购营销分组，如下图   


右侧操作中：

**编辑：**可以将添加门店时的信息进行编辑修改。


**商品管理：**进入后可进行商品添加操作。

       **添加商品：**一次可以选择一件商品进行添加，并且可以设置内购价，内购券使用数可以控制使用内购价购买该商品需要多少张内购券。（如下图）


       **会员管理：**可以查看当前内购营销活动下的参与会员详情以及可以调整其手中的内购券数量。
