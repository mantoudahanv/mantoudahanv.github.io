# 三.订单

# 3.1订单管理

# 3.1.1所有订单

点击“所有订单”进入到订单管理页面。所有的订单状态均可在这里查看。
下面来看一下有关的查询条件，我们可以根据以下查询条件快速的查找到你想要找到的相关订单。如下图所示：

点击选项卡，选择每页显示订单数量，可以查看相关状态下的订单情况。如下图所示：


订单列表中，可以清楚的查看订单的详细信息、发货单的打印、修改地址等操作，可以设置备注信息并展示备注信息，具体如下图所示：



选择 标记发货时，自提地址可以通过省市区筛选，便于门店较多的客户选择


# 3.1.1.1 标记发货可以选择是否分单发货

订单可以进行分单发货，一笔订单可以有多个物流单号。
如果需要使用分单发货和一个订单多物流单号，小程序版本需要更新到版本v9.0及以上
操作步骤：
总部后台→订单→订单管理→所有订单：订单可以选择分单发货，一笔订单可以有多个物流单号，同时手机端会相应生成不同的包裹。只有订单商品全部发完，订单状态才会变为待收货。（注：当供应商发货的开关开启时，订单中的供应商商品将不在分单发货中显示，供应商商品只能由供应商进行发货操作）

# 3.1.1.2 分单发货：分单发货选择“是”

标记发货弹框，分单发货选择“是”，可以选择多个商品发一包裹，或者一件商品数量太多，可以选择新增物流，添加多个物流，小程序端会分多个包裹展示，如图：

分单发货的订单，订单中所有商品发货完之后，才可以修改发货：

分单发货的订单，订单中所有商品发货完之后，才可以查看物流信息：

手机端根据发货物流展示多个包裹，每个包裹中的多个物流信息，可以切换包裹，切换物流查看包裹信息和物流信息：

# 3.1.2导出订单

商品导出到一个订单，开启后，订单导出时，多个订单商品显示到一个订单里面，可以设置订单类型、导出类型、过滤已关闭订单、付款完成时间、发货时间、订单创建时间、交易完成时间、导出内容等，进行订单导出。
①可以选择导出哪些活动的订单，比如只导出“秒杀活动订单”或非活动的订单；
②可以选择导出哪些内容，如跨境购客户，想要把发货订单发给保税区发货，只要收件人等相关字段就可以了。


# 3.1.3导出身份证图片

操作步骤：
启小铺后台→订单，增加导出身份证导出，商家可以根据自身需求设置导出类型、商品货号、付款时间等。

导出的身份证图片文件夹以订单号命名，每个文件夹显示身份证正、反面两张图片

# 3.1.5批量发货

发货后，请在ERP系统内筛选当天来源于本店铺的已发货订单，并导出表格。 请编辑导出的表格，只保留三个字段：订单号、物流公司、快递单号，再上传文件即可。

# 3.3售后服务

# 3.3.1退换货审核

所有关于订单的退换货都可以在此查看，处理相关的售后问题。如下图所示：

# 3.3.2商品评价

相关订单的评价都在此处查看，对于差评用户可以点击“隐藏”按钮进行隐藏。隐藏后仅评价方自己能看到，其它人均看不到。对于回复的内容可以点击“删除”按钮进行删除。

这里讲一下评价上传图片，手机端买家评价商品时，一个订单中有多个商品，可以逐个添加评价，可以上传图片，最多可以上传5张，在评价页面可以看到图片，也可以预览。

评价管理页面可以看到买家评价对应商品提交的图片。

# 3.3.3自定义评价

自定义评价，管理员可根据自已的需求可以添加自定义评价，上传相关图片，可选择评价的相关时间。点击“隐藏”按钮可以隐藏本条评价，可以根据商品名称和评价时间进行查询。

# 3.3.4商品评价导入

点击模板下载可以下载模板，然后按照模板规格上传评价。

# 3.3.6退换货设置

启小铺后台--订单--售后服务--退换货设置

商家可以选择会员申请退款时，是退到账户余额，还是微信账户，选择退到微信账户时，确保微信证书是否上传，是否上传正确。
①不开启商家审核，还是走原来的退换货流程不变
②退货物流单号是否必填只是针对原来的流程，如果开启，在提交退换货的时候，必须要填写物流单号，不开启不必填
着重说一下开启商家审核流程：
换货的流程是一样的，以退货流程为例：
（1）会员提交订单，商家发货后，会员申请退货，会员提交申请后，需要商家审核，才能继续之后流程的操作，在申请进度可以查看订单是等待商家审核，如图：

（2）退货申请提交后，要等待商家审核通过，如图：



（3）商家同意申请之后，申请进度可以查看申请状态：等待商家处理退货申请，可以提交快递单号，如图：

2.商家处理同意退货，订单关闭




**是否允许退款：**默认允许退款；开启后，将对会员隐藏退款按钮。
开启是否允许退款时，用户在申请售后页面中会将“仅退款”“退货退款”进行隐藏。
