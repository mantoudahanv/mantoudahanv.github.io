# 微信小程序代码上传操作步骤

此文档将会介绍微信小程序上传代码相关操作，关于开发工具下载，代码下载,代码导入,代码编译,提交审核这五个方面进行大致的阐述

在进行代码上传操作请注意,如果是授权第三方的,请现在微信小程序公众号后台进行停止授权操作,请找 到设置->第三方设置,如下图

# 1.工欲善其事,必先利其器

先下载微信小程序的开发者工具,工具的下载地址为: developers.weixin.qq.com/miniprogram/dev/devtools/stable.html点击跳转 点击进入之后找到开发->开发者工具->开发者工具更新及下载,下载稳定版最新的开发者工具,根据自己的系统进行选择(XP系统不支持)

# 2.下载完毕

下载完毕之后找到安装包进行安装,安装成功之后打开展示如下图,安装成功之后请使用小程序的管理员的微信进行扫码登录,打开之后默认选择为小程序/小游戏

# 3.登录商家后台

登录商家后台进入到设置->小程序设置进行代码包下载,如图所示:

# 4.下载代码包

下载下来代码包之后进行解压,解压之后如图展示,文件名称为店铺ID

# 5.打开微信小程序开发者工具

打开微信小程序开发者工具,选择导入,将下载并解压过的代码包进行导入

# 6.导入代码包

选择刚刚解压的代码包进行导入

项目名称可以自己定义,AppID在为自己后台授权的小程序的AppID

# 7.自动填充AppID

代码包选择之后,项目的名称以及AppID会自动填充(首先是小程序在商家后台授权过)

# 8.上传代码包

点击导入之后,等待页面加载完毕之后,看下页面是否正常加载以及加载的内容是否是自己商城的内容,如果确认正确无误之后,请点击上传

# 9.复制版本号

点击上传之后版本号可以在商家后台复制,例如第三步图,项目备注可以如例所示,根据当前日期走,然后点击上传,等待上传成功!成功之后登录微信小程序公众号后台mp.weixin.qq.com点击进入

# 10.提交审核

登录后台之后,选择版本管理,可以看到我们刚刚上传的版本,然后点击审核

# 11.同意审核规则

点击审核选择同意,然后进行下一步操作

# 12.继续提交

继续点击同意选择下一步

# 13.提交审核

以下信息可以不做更改,请点击提交审核

# 14.等待审核结果

提交完成之后,微信会在七个工作日内给出审核结果,到时候自己可以进行发布

审核结果会推送到微信公众平台管理员微信号内
