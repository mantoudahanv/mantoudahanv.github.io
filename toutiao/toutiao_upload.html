# 字节跳动小程序代码上传操作步骤

此文档将会介绍字节跳动小程序上传代码相关操作，关于开发工具下载，代码下载,代码导入,代码编译,提交审核这五个方面进行大致的阐述

# 1.工欲善其事,必先利其器

先下载字节跳动小程序的开发者工具,工具的下载地址为: microapp.bytedance.com/dev/miniapp/uYTNx4iN1EjL2UTM 点击进入之后找到开发->开发者工具->开发者工具更新及下载,下载最新的开发者工具,根据自己的系统进行选择

# 2.登录字节跳动开发者工具

下载完毕之后找到安装包进行安装,安装成功之后打开展示如下图,打开之后默认选择为小程序/小游戏,然后右侧输入字节跳动小程序开发者账号,进行登录

# 3.下载代码包

登录商家后台进入到字节跳动->小程序设置进行代码包下载,如图所示:

下载代码包之前小程序的配置必须完成,否则小程序将无法打开

# 4.解压代码包

下载下来代码包之后进行解压,解压之后如图展示,文件名称为店铺ID

# 5.导入代码包

打开字节跳动小程序开发者工具,选择导入,将下载并解压过的代码包进行导入

# 6.选择代码包

点击导入之后选择解压出来的代码包,然后点击打开,即可进入小程序

# 7.预览代码包

进入小程序页面加载完成之后展示如下,可以进行点击切换页面看页面加载是否正常,没有问题即可点击上传按钮进行代码编译上传

# 8.上传代码包

等小程序加载之后,看到是自己商城的信息,就可以去点击上传,填写版本号,填写更新日志示例如下,版本号可以在商家后台字节跳动->小程序设置中获得(图示如:第3步图)

# 9.等待编译结束

点击确认之后开发者工具将开始对代码进行编译处理,请耐心等待

# 10.完成上传

编译成功之后点击确定,然后可以关闭开发者工具,如下图则表示代码已经上传成功然后去字节跳动开发者后台进行版本提审操作

# 11.登录字节开发者平台

登录字节开发者后台,选择快捷登录或者直接进入开发者平台都可以进行登录进入开发者后台点击跳转

# 12.检查上传的代码包

进入当前小程序管理界面点击开发栏目即可看到刚刚自己的上传的版本号,用头条APP或者抖音APP扫码体验

# 13.提交审核

确保信息无误的情况下点击提审按钮,进行提交审核,字节提交审核需要如下操作,请如实添加,然后点击下一步:

# 14.勾选审核设置

字节跳动小程序自审选择,需要每一项进行打勾,弹出层左下角可以选择勾选,在审核通过之后,可以直接发布,点击选择完之后,点击提审,图示如下:

# 15.然后可以看到提审后的页面

# 16.等待审核通过

提交审核之后,审核需要1-7个工作日,商家可以在字节开发者后台自行查看,如有审核失败,根据审核失败的原因进行处理
